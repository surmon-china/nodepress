### Development

```bash
$ pnpm install

# dev
$ pnpm run start:dev

# test
$ pnpm run lint
$ pnpm run test
$ pnpm run test:e2e
$ pnpm run test:cov
$ pnpm run test:watch

# build
$ pnpm run build

# run
$ pnpm run start:prod
```

### Actions

- Any PR open → `CI:Build test`
- New tag `v*` → `CI:Create release`
- Release created → `CI:Deploy` → `CI:Execute server deploy script`
