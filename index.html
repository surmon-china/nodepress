NodePress

       

RESTful API service for surmon.me blog, powered by NestJS, required MongoDB & Redis.

适用于 surmon.me 的 RESTful API 服务端应用；基于 NestJS 开发，需安装 MongoDB 和 Redis 方可完整运行。

Development

Example :
$ pnpm install

# dev
$ pnpm run start:dev

# test
$ pnpm run lint
$ pnpm run test
$ pnpm run test:e2e
$ pnpm run test:cov
$ pnpm run test:watch

# build
$ pnpm run build

# run
$ pnpm run start:prod

Actions

  • Any PR open → CI:Build test
  • New tag v* → CI:Create release
  • Release created → CI:Deploy → CI:Execute server deploy script
